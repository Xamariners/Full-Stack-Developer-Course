XAM140 Resources and Styles

Exercise 2: Use page-level resources

In this exercise, you will use page-level Resources to eliminate repeated values in XAML.

To complete the exercise, you will need Visual Studio for Windows or macOS with the Xamarin development tools installed. You will also need either an emulator/simulator or a device to run the exercise on. Please see the setup page if you need help installing the Xamarin development environment.

Open the starter solution

This exercise is a continuation of the previous exercise. You can use your existing solution or begin from the prior Exercise 1 > Completed solution in your copy of the cloned or downloaded course materials.

If you're using Visual Studio on Windows, you'll see all of the platform or head projects: iOS, Android and Windows. If you're using macOS, the Windows projects will be disabled (grayed out).

Add a resource dictionary

Let's create a resource dictionary in XAML.

  1. Open the TipCalculator project.
  2. Open StandardTipPage.xaml.
  3. Create a ResourceDictionary in the Page's Resources property.
<Page.Resources>
  <ResourceDictionary>
  </ResourceDictionary>
</Page.Resources>

Define color resources

  1. Define a Color resource inside the resource dictionary. Give it an x:Key of "bgColor" and a value of #c0c0c0.
  2. Define a second Color resource inside the resource dictionary. Give it an x:Key of "fgColor" and a value of #606060.
<Page.Resources>
  <ResourceDictionary>
    <Color x:Key="bgColor">#c0c0c0</Color>
    <Color x:Key="fgColor">#606060</Color>
  </ResourceDictionary>
</Page.Resources>

Use static resources

Now, let's apply the Resources you just created.

  1. Use the StaticResource markup extension to apply the bgColor resource to the Background property of the page's root Grid.
  2. Use the StaticResource markup extension to apply the fgColor resource to the TextColor property of all the Labels that use a TextColor of Navy.
<Grid x:Name ="LayoutRoot" BackgroundColor="{StaticResource bgColor}" Padding="10">
<Label x:Name="tipOutput" Text="0.00" TextColor="{StaticResource fgColor}" FontSize="22" Grid.Row="1" Grid.Column="1" />

Run the application

Run the application on a platform of your choice and verify that the StandardTipPage displays dark text on a light background at startup.
