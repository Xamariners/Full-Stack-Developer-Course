Exercise 4: Create and apply a style

The goal of this exercise is to define and apply a page-level style in your Xamarin.Forms application.

To complete the exercise, you will need Visual Studio for Windows or macOS with the Xamarin development tools installed. You will also need either an emulator/simulator or a device to run the exercise on. Please see the setup page if you need help installing the Xamarin development environment.

Open the starter solution

This exercise is a continuation of the previous exercise. You can use your existing solution or begin from the prior Exercise 3 > Completed solution in your copy of the cloned or downloaded course materials.

Define a style

Here you will implement a Style and store it in a page-level dictionary.

  1. Open the TipCalculator project.
  2. Open StandardTipPage.xaml.
  3. Add a Style to the page's resource dictionary. Use a Key of infoLabelStyle and a TargetType of Label.
<Style x:Key="infoLabelStyle" TargetType="Label"></Style>

Add a setter to your style

  1. Add a Setter to your Style that sets the FontSize property to 22.
  2. Add a Setter to your Style that sets the FontAttributes property to Bold.
<Style x:Key="infoLabelStyle" TargetType="Label">
    <Setter Property="FontSize" Value="22" />
    <Setter Property="FontAttributes" Value="Bold" />
</Style>

Apply the style

Now, you will apply the style to the appropriate elements in your XAML.

  1. Locate the three Labels that use TextSize of 22 and FontAttributes of Bold. Remove those property assignments from the labels.
  2. Use the StaticResource markup extension to assign the Style that you just defined to the Style property of the three labels. Sample code for one Label is provided:
<Label Style="{StaticResource infoLabelStyle}" â€¦ />

Run the application

Run the app on a platform of your choice. The app should look exactly like it did before, however now the font attributes are being set with Styles.

Change the font styles

Let's take a look at how easy it is to update the styles now that they have been created.

  1. Go back to your style in the resource dictionary and change the font size to 32.
  2. Run the app again to view the changes.

Exercise summary

In this exercise, you replaced individual property settings with a Style. This reduced the clutter of multiple property settings on each control and made the XAML easier to read.
